---

<h2> 🚀 Features </h2>

✅  Displays a **random quote** on page load.  
✅  Click **"New Quote"** to get a new random quote.  
✅  Uses **thousands of real quotes** from famous authors.  
✅  Styled with a **Windows 11-inspired UI**.  
✅  **Mobile-friendly** and responsive design.
